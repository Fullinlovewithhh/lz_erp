# 业务单据与状态索引

> 文档性质：跨模块单据索引
>
> 说明：本文件定义单据归属和关键状态，不代替数据库字段字典。

## 1. 客户与订单阶段

| 业务单据 | 归属层级 | 作用 | 关键状态或规则 |
|---|---|---|---|
| 客户档案 | 客户 | 保存客户主体、联系方式和政策 | 启用、停用；不因订单结束删除 |
| 客户订单 | 客户 | 客户本次来单的业务汇总载体 | 客户订单号唯一；可暂时没有工厂订单 |
| 客户原始资料 | 客户订单 | 保存 CAD 及客户后续修改版本 | 版本追加，不覆盖历史文件 |
| CAD评审领取记录 | 客户订单 | 记录哪位深化工程师领取 | 待领取、已领取、已完成拆单 |
| 拆单草稿 | 客户订单 | 描述拟生成的工厂订单 | 草稿、已提交；提交后正式生成工厂订单 |
| 工厂订单 | 客户订单 | 报价、下料和生产的执行单元 | L、V、补单或售后；编号创建后不可修改 |

## 2. 报价阶段

| 业务单据 | 归属层级 | 作用 | 关键状态或规则 |
|---|---|---|---|
| 报价任务 | 工厂订单 | 记录当前负责人和转交 | 待领取、处理中、已提交；历史转交永久保留 |
| 工厂订单报价 | 工厂订单 | 保存一张当前报价及历史版本 | 每次修改形成版本，只有一个当前有效版本 |
| 报价明细 | 报价版本 | 产品、尺寸、数量、单位和金额 | 产品价格来自产品资料快照 |
| 特殊加价 | 报价明细 | 常用规则或一次性特殊费用 | 可参与或不参与折扣，必须保留来源说明 |
| 五金报价明细 | 报价版本 | 加入报价总额的五金项目 | 默认不参与工厂订单折扣 |
| 临时折扣申请 | 工厂订单 | 固定政策以外的折扣审批 | 待审批、通过、驳回、失效 |
| 客户订单成交价调整 | 客户订单 | 保存议价、抹零及厂长审批 | 待厂长审批、已批准、已驳回、已失效 |
| V外部报价登记 | V工厂订单 | 保存板式外部系统报价结果 | 登记、客户确认、失效；按版本留痕 |
| L客户报价PDF | 客户订单 | 汇总同一客户订单下的全部 L 报价 | 待客户确认、客户已确认、已失效 |
| L客户报价确认 | 客户订单 | 记录客户对全部 L 工厂订单的整单确认 | 确认方式、时间和记录人必填 |
| V外部报价确认 | V工厂订单 | 记录客户对板式外部报价的确认 | 与外部报价版本一起留痕 |

## 3. 状态联动原则

- 拆单提交后，L 工厂订单进入 ERP 报价池，V 工厂订单进入外部报价登记流程。
- L 报价修改或 L 成交价变化，会导致旧 L 客户 PDF 和旧 L 确认失效。
- V 外部报价修改会使旧 V 确认失效，但不要求重做未发生变化的 L 报价 PDF。
- 只有 L、V 各自满足报价和客户确认条件后，客户订单才能形成当前有效的统一财务应收。
- 报价完成后仍允许修改，但不得删除历史版本和已经确认的到账记录。
- 客户明确不做时保留订单、CAD、拆单和报价历史，以备注说明，不进入付款与生产。

## 4. 付款与放行阶段

| 业务单据 | 归属层级 | 作用 | 关键状态或规则 |
|---|---|---|---|
| 付款计划 | 客户订单 | 保存全款、分期、月结或免单安排 | 草稿、待审批、已批准、已变更、已完成；按版本留痕 |
| 付款计划期次 | 付款计划版本 | 保存每期金额和到期日 | 待收、部分到账、已结清、已逾期 |
| 到账核对申请 | 客户订单 | 客服登记客户声称已经付款 | 待财务确认、已确认、已驳回 |
| 财务到账记录 | 客户订单 | 保存真实到账和收款凭证 | 有效、已冲正、已作废；确认后不得覆盖 |
| 到账冲抵记录 | 付款期次 | 记录实际到账如何冲抵各期 | 优先冲抵最早未结清期次 |
| 下料放行申请 | 客户订单 | 请求整张客户订单取得下料资格 | 待厂长审批、已批准、已驳回、已失效 |
| 客服放行记录 | 客户订单 | 记录客服执行整单放行 | 待客服放行、已放行、已撤回 |
| 月结客户状态 | 客户 | 控制客户级月结和逾期限制 | 正常、逾期锁定、例外批准、已解除 |

关键联动：

- 付款计划审批和下料放行审批是两个独立审批。
- 客服提交付款通知不计入到账，只有财务确认后才增加累计到账。
- 报价、付款计划、到账有效性或客户确认发生关键变化时，原下料审批失效。
- 下料放行绑定客户订单并包含当时全部有效工厂订单。
- 厂长可以在下料开始前或开始后撤回放行，原因和现场停止位置必须留痕。

## 5. 后续待补单据

以下单据已经有业务方向，但应在对应流程确认后再写入正式状态：

- BOM、板材清单、五金锁库和采购申请。
- 生产任务、岗位工期、工序扫码、质检和包装记录。
- 售后申请、责任认定和重做范围审批。
