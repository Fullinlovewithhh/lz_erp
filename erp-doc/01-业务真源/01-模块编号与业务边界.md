# ERP 模块编号与业务边界

> 文档性质：页面模块索引，不代替具体业务流程。
>
> 本文固定当前 ERP 左侧菜单编号。后续需求、Git 分支和提交说明可以直接引用编号，例如“修改 3.1 客户订单”。

## 1. 基础资料

| 编号 | 模块 | 说明 |
|---|---|---|
| 1.1 | 人员管理 | 员工和岗位资料 |
| 1.2 | 账号权限 | 登录账号、角色和权限 |
| 1.3 | 产品资料 | 报价和生产使用的产品资料 |
| 1.4 | 材料资料 | 板材、辅料等材料资料 |
| 1.5 | 工艺规则 | 工艺与特殊加工规则 |
| 1.6 | 价格规则 | 基础价和加价规则 |
| 1.7 | 生产线配置 | 生产线名称、编号前缀、启停和排序 |

## 2. 客户管理

项目层已经取消，客户下面直接建立客户订单。

| 编号 | 模块 | 说明 |
|---|---|---|
| 2.1 | 客户档案 | 客户名称、联系方式、地址和状态 |
| 2.2 | 跟进记录 | 客户沟通和回访记录 |
| 2.3 | 客户需求 | 客户需求和备注 |

## 3. 订单管理

| 编号 | 模块 | 说明 |
|---|---|---|
| 3.1 | 客户订单 | 选择客户、填写客户订单号并上传原始 CAD |
| 3.2 | CAD评审池 | 深化工程师领取客户 CAD |
| 3.3 | 拆单 | 深化工程师拆成一张或多张工厂订单并直接提交报价池 |
| 3.4 | 工厂订单 | 查看正式工厂订单、生产线和业务状态 |
| 3.5 | 补单管理 | 管理关联原工厂订单的补单 |
| 3.6 | 售后管理 | 暂定入口，待售后业务确认后开发 |

## 4. 报价管理

| 编号 | 模块 | 说明 |
|---|---|---|
| 4.1 | 报价订单池 | 未指定负责人或等待领取的工厂订单 |
| 4.2 | 完整报价明细 | 产品、材料、尺寸、工艺、五金和价格 |
| 4.3 | 报价版本 | 工厂订单报价版本和修改历史 |
| 4.4 | 客户报价汇总 | 汇总客户订单下的全部工厂订单报价 |
| 4.5 | 报价PDF与客户确认 | 生成PDF并记录客户确认结果 |

## 5. 排单管理

| 编号 | 模块 | 说明 |
|---|---|---|
| 5.1 | 待下料订单 | 已满足付款和审批条件的客户订单 |
| 5.2 | 板材清单 | 下料生成的板件需求 |
| 5.3 | 五金清单 | 下料生成的五金需求 |
| 5.4 | BOM清单 | 正式生产物料和结构清单 |

## 6. 生产管理

| 编号 | 模块 | 说明 |
|---|---|---|
| 6.1 | 生产任务 | 下发正式生产任务 |
| 6.2 | 排产计划 | 安排生产线、日期、设备和人员 |
| 6.3 | 工序进度 | 记录各生产工序进度 |
| 6.4 | 质检包装 | 质检、返工、包装和完工 |

## 7. 采购库存

| 编号 | 模块 | 说明 |
|---|---|---|
| 7.1 | 五金资料 | 五金报价、采购和库存基础资料 |
| 7.2 | 库存管理 | 实际库存、预留库存和可用库存 |
| 7.3 | 采购申请 | 缺料后生成采购需求 |
| 7.4 | 采购订单 | 供应商下单和到货管理 |
| 7.5 | 出入库记录 | 入库、领料、退料和出库流水 |

## 8. 财务数据

| 编号 | 模块 | 说明 |
|---|---|---|
| 8.1 | 付款计划 | 全款、分期、月结和付款到期日 |
| 8.2 | 到账确认 | 财务记录并确认实际到账 |
| 8.3 | 价格调整审批 | 厂长审批成交价调整 |
| 8.4 | 下料放行 | 财务确认、厂长审批和客服下料放行 |
| 8.5 | 月结与逾期 | 客户级月结逾期和业务锁定 |
| 8.6 | 收款账户配置 | 可维护的收款方式和账户 |

## 9. 当前边界

- 3.1 至 3.5、4.x 和 8.x 至客服下料放行已经形成正式业务文档；3.6 售后仍待确认。
- 5.x、6.x 和 7.x 尚未形成经过确认的完整生产闭环。
- 项目入口和正式项目关联已取消。
- 旧表和旧接口的实现状态统一记录在《遗留兼容说明》，不作为模块业务规则。
